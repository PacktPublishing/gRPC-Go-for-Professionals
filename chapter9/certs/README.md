# Certs

The files in here are copied from the x509 folder in the [grpc-go/examples/data](https://github.com/grpc/grpc-go/tree/master/examples/data/x509) folder in [grpc-go](https://github.com/grpc/grpc-go). This lets us focus on writing gRPC code and not worrying generating these certificates (let the experts handle this).
